# Loan Approval Prediction System

## 📌 Project Overview

This project predicts whether a loan application will be approved or rejected using machine learning techniques based on applicant details like income, loan amount, credit history, etc.

## 🚀 Features

- Data preprocessing and cleaning
- Machine learning model (Random Forest)
- Prediction system for loan approval

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
