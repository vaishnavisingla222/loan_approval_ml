# 🏦 Loan Approval Prediction System

## 🧠 Problem Statement

Banks receive thousands of loan applications. Evaluating them manually is time-consuming and error-prone.
This project uses Machine Learning to predict whether a loan will be **Approved or Rejected** based on applicant details.

---

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing
- 🤖 Model Training & Comparison
- 📈 Feature Importance Analysis
- 🖥️ Streamlit Web App for predictions

---

## ⚙️ Tech Stack

- Python 🐍
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Streamlit

---

## 📊 Dataset

The dataset contains financial and demographic details of loan applicants:

- Dependents
- Education
- Self-employed
- Income
- Loan amount
- Loan term
- CIBIL score
- Asset values

---

## 🔄 Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Encoding & Scaling
4. Model Training
5. Model Comparison
6. Final Model Selection
7. Model Saving
8. Streamlit App Development

---

## 🤖 Models Used

- Logistic Regression
- Decision Tree
- Random Forest ✅ (Final Model)

---

## 📈 Results & Insights

- CIBIL score is the most important feature
- Higher income and assets increase approval chances
- Random Forest gave the best performance

## 🖥️ Run Locally

### Clone repo

```bash
git clone https://github.com/vaishnavisingla222/loan_approval_ml.git
cd loan_approval_ml
```

### Create environment

```bash
python -m venv loan_env
source loan_env/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run app

```bash
streamlit run app.py
```
